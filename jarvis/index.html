JARVIS

Personal AI Executive Assistant

A personal command center built to manage the complexity of career, business, and life — simultaneously.

Claude API Node.js Plaid Notion Gmail MCP

Overview

One system. Everything connected.

JARVIS is a personal AI command center designed and built to manage the complexity of running a career, two businesses, and a family's financial future simultaneously. The system integrates email, calendar, finances, and active projects into a unified intelligence layer — surfacing daily briefings, decision support, and real-time context.

Built on Claude (Anthropic) as the AI layer and designed to be model agnostic, with architecture intentionally decoupled from any single provider.

1 Unified intelligence layer
6+ Integrated data sources
∞ Model agnostic by design

Key Capabilities

What JARVIS does

Daily Briefings

Automated daily briefings synthesized from email, calendar, and financial data — surfacing what matters before the day begins.

Project Intelligence

Tracks priorities, deadlines, and action items across work and business initiatives — maintaining context across every active program.

Business Dashboards

Live dashboards for Robinson Consulting Group and Robinson Business Ventures — turning operational data into actionable intelligence.

Financial Monitoring

Monitors investment accounts, credit, and debt payoff tracking across all accounts — keeping financial goals in view at all times.

AI Decision Support

Context-aware decision support grounded in personal and professional history — not a generic chatbot, but a system that knows the situation.

Stack & Tools

Built with

Claude API

Anthropic — AI reasoning, briefing generation, and decision support layer

Node.js

Backend runtime — orchestrating integrations, scheduling, and data pipelines

Plaid

Financial data layer — account balances, transactions, and investment tracking

Gmail + Google Calendar

Communication and scheduling context — the raw signal for daily briefings

Notion

Knowledge base and project management — the persistent memory layer

MCP Integrations

Model Context Protocol — extensible tooling for connecting Claude to live data sources
