Overview

One system. Everything connected.

JARVIS is a personal AI command center I designed and built to manage the complexity of running a career, two businesses, and a family's financial future at once. It integrates email, calendar, finances, and active projects into a unified intelligence layer — surfacing daily briefings, decision support, and real-time context.

It's built on Claude (Anthropic) as the reasoning layer and designed to be model-agnostic, with the architecture intentionally decoupled from any single provider.

02

Capabilities

Daily briefings
Automated briefings synthesized from email, calendar, and financial data — surfacing what matters before the day begins.
Project intelligence
Tracks priorities, deadlines, and action items across work and business initiatives, maintaining context across every active program.
Business dashboards
Live dashboards for Robinson Consulting Group and Robinson Business Ventures, turning operational data into actionable intelligence.
Financial monitoring
Monitors investment accounts, credit, and debt-payoff tracking across accounts — keeping financial goals in view at all times.
AI decision support
Context-aware support grounded in personal and professional history — not a generic chatbot, but a system that knows the situation.
03

Stack

Claude API
Anthropic — reasoning, briefing generation, and the decision-support layer.
Node.js
Backend runtime — orchestrating integrations, scheduling, and data pipelines.
Plaid
Financial data layer — balances, transactions, and investment tracking.
Gmail + Google Calendar
Communication and scheduling context — the raw signal for daily briefings.
Notion
Knowledge base and project management — the persistent memory layer.
MCP integrations
Model Context Protocol — extensible tooling connecting Claude to live data sources.
